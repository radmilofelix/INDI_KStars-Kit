INDI_KStars-Kit

INDI and KStars build/install script

Clone the repository: _**git clone https://github.com/radmilofelix/INDI_KStars-Kit**_

Change directory to the repository: _**cd INDI_KStars-Kit**_

Give execute permissions to the script: _**chmod 775 INDI_KStars-Kit.sh**_

The script has to be run as root.

Run _**./INDI_KStars-Kit.sh UpdateUsefulFunctions**_ to get the useful functions bash script

To build INDI & KStars run: _**./INDI_KStars-Kit.sh BuildAll**_

To install KStars and build INDI (only on Ubuntu sustems) run: **_./INDI_KStars-Kit.sh InstallAll**_

The script will also install an USB reset script and an UDEV rules file for the devices connected bt USB.
