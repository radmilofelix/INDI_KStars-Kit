# INDI_KStars-Kit
INDI and KStars build/install script
Clone the repository: git clone https://github.com/radmilofelix/INDI_KStars-Kit
Change directory to the repository: cd INDI_KStars-Kit
Give execute permissions to the script: chmod 775 INDI_KStars-Kit.sh
Script has to be run as root.
Run ./INDI_KStars-Kit.sh UpdateUsefulFunctions to get the useful functions bash script
To build INDI & KStars run: ./INDI_KStars-Kit.sh BuildAll
To install KStars and build INDI (only on Ubuntu sustems) run: ./INDI_KStars-Kit.sh InstallAll
The script will also install an USB reset script and an UDEV rules file for the devices connected bt USB.
